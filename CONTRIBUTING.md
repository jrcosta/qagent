# Contribuindo para o QAgent

Obrigado pelo seu interesse em contribuir!

## Como contribuir

1. Fork este repositório e crie sua branch a partir de `main`.
2. Faça suas alterações seguindo as diretrizes do projeto (veja `AGENTS.md`).
3. Garanta que todos os testes estejam passando.
4. Abra um Pull Request detalhando sua contribuição.

## Boas práticas
- Siga o padrão de mensagens de commit do Conventional Commits.
- Documente mudanças relevantes no README ou na documentação.
- Não inclua segredos ou credenciais no código.
- Prefira abrir issues para discutir grandes mudanças antes de implementá-las.

## Código de Conduta
Este projeto adota um código de conduta para garantir um ambiente acolhedor e respeitoso para todos.
